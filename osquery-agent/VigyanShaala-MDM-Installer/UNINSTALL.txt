========================================
How to Uninstall VigyanShaala MDM Agent
========================================

If you need to remove the osquery agent from this computer:

METHOD 1: Using the Uninstaller (Easiest)
------------------------------------------
1. Right-click on "UNINSTALL.bat"
2. Select "Run as Administrator"
3. Confirm uninstallation (type Y)
4. Wait for completion
5. Done!

METHOD 2: Manual Uninstallation
--------------------------------
1. Open PowerShell as Administrator
2. Stop the service:
   Stop-Service osqueryd
3. Remove the service:
   C:\Program Files\osquery\osqueryd.exe --uninstall
4. Uninstall via Windows Settings:
   - Go to Settings > Apps > Apps & features
   - Find "osquery" and uninstall
5. Delete installation folder:
   C:\Program Files\osquery
6. Delete ProgramData folder:
   C:\ProgramData\osquery

Note: The device entry in the dashboard will remain.
      Contact administrator if you want it removed from the dashboard.
